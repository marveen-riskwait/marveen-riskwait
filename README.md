## 🌟 Featured project — SideQuest

> A mobile-first social platform to discover and organize real-world events.
> Built with a team in Agile sprints — **React · Flask · PostgreSQL**.

- **55 REST endpoints** across 9 SQLAlchemy models, Alembic migrations
- **JWT auth** in httpOnly cookies, email verification & password recovery
- **Realtime chat & notifications** via Socket.IO
- **Leaflet map** with live geolocation, Ticketmaster API via cached proxy
- **Cloudinary** uploads, friends & invitations system
- **Dockerized** on Fly.io with gunicorn

🔗 **[sidequest-beta.fly.dev →](https://sidequest-beta.fly.dev)**
